# Heap Basics

Heaps are complete binary trees stored efficiently in arrays.

## Array Representation

```
Index:    0   1   2   3   4   5   6
         [50, 30, 20, 15, 10, 8, 16]

              50
            /    \
          30      20
         /  \    /  \
        15  10  8   16

Parent of i: (i - 1) / 2
Left child: 2 * i + 1
Right child: 2 * i + 2
```

## Max Heap Implementation

```c
#define MAX_SIZE 100

typedef struct {
    int items[MAX_SIZE];
    int size;
} MaxHeap;

void initHeap(MaxHeap *h) {
    h->size = 0;
}

int isHeapEmpty(MaxHeap *h) {
    return h->size == 0;
}

int isHeapFull(MaxHeap *h) {
    return h->size == MAX_SIZE;
}

// Get parent index
int parent(int i) {
    return (i - 1) / 2;
}

// Get left child index
int leftChild(int i) {
    return 2 * i + 1;
}

// Get right child index
int rightChild(int i) {
    return 2 * i + 2;
}
```

## Swap Function

```c
void swap(MaxHeap *h, int i, int j) {
    int temp = h->items[i];
    h->items[i] = h->items[j];
    h->items[j] = temp;
}
```
