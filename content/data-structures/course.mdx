# Data Structures

Welcome to the comprehensive data structures course! This course features **12 in-depth lessons** and **2 hands-on projects** with **70 total steps**, taking you from fundamental data organization to advanced structures through practical implementation.

## Course Structure

### Foundations (Lessons 1-4)

- **Lesson 1**: Arrays and Strings (6 steps)
- **Lesson 2**: Linked Lists (6 steps)
- **Lesson 3**: Stacks (5 steps)
- **Lesson 4**: Queues (5 steps)

### Hierarchical Structures (Lessons 5-7)

- **Lesson 5**: Trees Introduction (6 steps)
- **Lesson 6**: Binary Search Trees (6 steps)
- **Lesson 7**: Heaps and Priority Queues (5 steps)

### Advanced Structures (Lessons 8-10)

- **Lesson 8**: Hash Tables (6 steps)
- **Lesson 9**: Graphs Basics (6 steps)
- **Lesson 10**: Graph Traversal (5 steps)

### Specialized Topics (Lessons 11-12)

- **Lesson 11**: Advanced Trees - AVL and Red-Black (5 steps)
- **Lesson 12**: Course Project - File System Implementation (10 steps)

## What You'll Learn

- **Linear Structures**: Arrays, linked lists, stacks, queues
- **Hierarchical Structures**: Trees, binary search trees, AVL trees
- **Graph Structures**: Graphs, adjacency lists, traversal algorithms
- **Hash-Based Structures**: Hash tables, collision handling
- **Heap Structures**: Priority queues, heap operations
- **Memory Management**: Dynamic allocation, pointer manipulation
- **Real-World Applications**: Database indexes, file systems, scheduling

## Key Features

- **C Implementation**: All data structures implemented in C for deep understanding
- **Hands-On Projects**: Build a file system and database index
- **Visual Learning**: Step-by-step diagrams and animations
- **Complexity Analysis**: Understand time and space trade-offs
- **Best Practices**: Memory safety, leak prevention, debugging techniques

## Prerequisites

- Completion of "Learning C" course or equivalent C programming experience
- Understanding of pointers, dynamic memory allocation
- Familiarity with basic algorithms (searching, sorting)

## Recommended Timeline

This course is designed for completion in **10-14 weeks**:

- **Intensive**: 2 lessons per week (6 weeks)
- **Standard**: 1 lesson per week (12 weeks)
- **Self-Paced**: Progress at your own speed
