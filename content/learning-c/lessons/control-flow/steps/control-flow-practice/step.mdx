# Control Flow Practice

Let's put everything together with practical programming challenges that combine conditional statements, loops, and control flow keywords.

## FizzBuzz

A classic programming challenge: Print numbers 1 to 100, but:
- For multiples of 3, print "Fizz"
- For multiples of 5, print "Buzz"
- For multiples of both, print "FizzBuzz"

```c
#include <stdio.h>

int main(void) {
    for (int i = 1; i <= 100; i++) {
        if (i % 3 == 0 && i % 5 == 0) {
            printf("FizzBuzz\n");
        } else if (i % 3 == 0) {
            printf("Fizz\n");
        } else if (i % 5 == 0) {
            printf("Buzz\n");
        } else {
            printf("%d\n", i);
        }
    }

    return 0;
}
```

Alternative approach:

```c
#include <stdio.h>

int main(void) {
    for (int i = 1; i <= 100; i++) {
        int printed = 0;

        if (i % 3 == 0) {
            printf("Fizz");
            printed = 1;
        }

        if (i % 5 == 0) {
            printf("Buzz");
            printed = 1;
        }

        if (!printed) {
            printf("%d", i);
        }

        printf("\n");
    }

    return 0;
}
```

## Number Guessing Game

Complete implementation with hints, attempts, and replay option.

```c
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    srand(time(NULL));
    char play_again;

    do {
        int secret = rand() % 100 + 1;
        int guess;
        int attempts = 0;
        int max_attempts = 7;

        printf("\n=== Number Guessing Game ===\n");
        printf("I'm thinking of a number between 1 and 100.\n");
        printf("You have %d attempts.\n\n", max_attempts);

        while (attempts < max_attempts) {
            attempts++;
            printf("Attempt %d/%d - Enter your guess: ", attempts, max_attempts);
            scanf("%d", &guess);

            if (guess < 1 || guess > 100) {
                printf("Please guess between 1 and 100!\n");
                attempts--;  // Don't count invalid guesses
                continue;
            }

            if (guess == secret) {
                printf("\n*** Congratulations! ***\n");
                printf("You found it in %d attempts!\n", attempts);
                break;
            } else if (guess < secret) {
                int diff = secret - guess;
                if (diff <= 5) {
                    printf("Too low, but very close!\n");
                } else if (diff <= 10) {
                    printf("Too low, getting warm!\n");
                } else {
                    printf("Too low!\n");
                }
            } else {
                int diff = guess - secret;
                if (diff <= 5) {
                    printf("Too high, but very close!\n");
                } else if (diff <= 10) {
                    printf("Too high, getting warm!\n");
                } else {
                    printf("Too high!\n");
                }
            }

            printf("Attempts remaining: %d\n\n", max_attempts - attempts);
        }

        if (guess != secret) {
            printf("\nGame Over! The number was %d.\n", secret);
        }

        printf("\nPlay again? (Y/N): ");
        scanf(" %c", &play_again);

    } while (play_again == 'Y' || play_again == 'y');

    printf("Thanks for playing!\n");

    return 0;
}
```

## Pattern Printing

### Number Pyramid

```c
#include <stdio.h>

int main(void) {
    int rows = 5;

    printf("Number Pyramid:\n");
    for (int i = 1; i <= rows; i++) {
        // Print spaces
        for (int j = 1; j <= rows - i; j++) {
            printf(" ");
        }

        // Print ascending numbers
        for (int j = 1; j <= i; j++) {
            printf("%d", j);
        }

        // Print descending numbers
        for (int j = i - 1; j >= 1; j--) {
            printf("%d", j);
        }

        printf("\n");
    }

    return 0;
}
```

Output:
```
    1
   121
  12321
 1234321
123454321
```

### Diamond Pattern

```c
#include <stdio.h>

int main(void) {
    int n = 5;

    printf("Diamond Pattern:\n");

    // Upper half
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n - i; j++) {
            printf(" ");
        }
        for (int j = 1; j <= 2 * i - 1; j++) {
            printf("*");
        }
        printf("\n");
    }

    // Lower half
    for (int i = n - 1; i >= 1; i--) {
        for (int j = 1; j <= n - i; j++) {
            printf(" ");
        }
        for (int j = 1; j <= 2 * i - 1; j++) {
            printf("*");
        }
        printf("\n");
    }

    return 0;
}
```

Output:
```
    *
   ***
  *****
 *******
*********
 *******
  *****
   ***
    *
```

### Hollow Rectangle

```c
#include <stdio.h>

int main(void) {
    int width = 10;
    int height = 5;

    printf("Hollow Rectangle:\n");
    for (int i = 1; i <= height; i++) {
        for (int j = 1; j <= width; j++) {
            // Print * on border, space inside
            if (i == 1 || i == height || j == 1 || j == width) {
                printf("*");
            } else {
                printf(" ");
            }
        }
        printf("\n");
    }

    return 0;
}
```

Output:
```
**********
*        *
*        *
*        *
**********
```

## Menu-Driven Calculator

```c
#include <stdio.h>
#include <math.h>

void display_menu(void) {
    printf("\n=== Calculator Menu ===\n");
    printf("1. Addition\n");
    printf("2. Subtraction\n");
    printf("3. Multiplication\n");
    printf("4. Division\n");
    printf("5. Power\n");
    printf("6. Square Root\n");
    printf("7. Exit\n");
    printf("Enter choice: ");
}

int main(void) {
    int choice;
    double num1, num2, result;

    do {
        display_menu();
        scanf("%d", &choice);

        if (choice == 7) {
            printf("Goodbye!\n");
            break;
        }

        if (choice < 1 || choice > 7) {
            printf("Invalid choice!\n");
            continue;
        }

        // Get input(s)
        if (choice == 6) {
            printf("Enter number: ");
            scanf("%lf", &num1);
        } else {
            printf("Enter first number: ");
            scanf("%lf", &num1);
            printf("Enter second number: ");
            scanf("%lf", &num2);
        }

        // Perform calculation
        switch (choice) {
            case 1:
                result = num1 + num2;
                printf("%.2lf + %.2lf = %.2lf\n", num1, num2, result);
                break;
            case 2:
                result = num1 - num2;
                printf("%.2lf - %.2lf = %.2lf\n", num1, num2, result);
                break;
            case 3:
                result = num1 * num2;
                printf("%.2lf * %.2lf = %.2lf\n", num1, num2, result);
                break;
            case 4:
                if (num2 == 0) {
                    printf("Error: Division by zero!\n");
                } else {
                    result = num1 / num2;
                    printf("%.2lf / %.2lf = %.2lf\n", num1, num2, result);
                }
                break;
            case 5:
                result = pow(num1, num2);
                printf("%.2lf ^ %.2lf = %.2lf\n", num1, num2, result);
                break;
            case 6:
                if (num1 < 0) {
                    printf("Error: Cannot take square root of negative number!\n");
                } else {
                    result = sqrt(num1);
                    printf("sqrt(%.2lf) = %.2lf\n", num1, result);
                }
                break;
        }

    } while (1);

    return 0;
}
```

## Prime Number Finder

Find all prime numbers up to N using the Sieve of Eratosthenes approach.

```c
#include <stdio.h>

int main(void) {
    int n;
    printf("Find all prime numbers up to: ");
    scanf("%d", &n);

    if (n < 2) {
        printf("No prime numbers less than 2.\n");
        return 0;
    }

    printf("Prime numbers up to %d:\n", n);

    for (int num = 2; num <= n; num++) {
        int is_prime = 1;

        // Check if num is divisible by any number from 2 to sqrt(num)
        for (int i = 2; i * i <= num; i++) {
            if (num % i == 0) {
                is_prime = 0;
                break;  // Found a divisor, not prime
            }
        }

        if (is_prime) {
            printf("%d ", num);
        }
    }

    printf("\n");

    return 0;
}
```

## ATM Simulator

```c
#include <stdio.h>

int main(void) {
    double balance = 1000.00;
    int choice;
    double amount;
    int pin = 1234;
    int entered_pin;
    int attempts = 0;

    // PIN verification
    do {
        printf("Enter PIN: ");
        scanf("%d", &entered_pin);

        if (entered_pin == pin) {
            break;
        }

        attempts++;
        printf("Incorrect PIN! ");

        if (attempts >= 3) {
            printf("Too many attempts. Card blocked.\n");
            return 0;
        }

        printf("Attempts remaining: %d\n", 3 - attempts);

    } while (1);

    printf("PIN accepted.\n");

    // Main ATM menu
    do {
        printf("\n=== ATM Menu ===\n");
        printf("1. Check Balance\n");
        printf("2. Deposit\n");
        printf("3. Withdraw\n");
        printf("4. Exit\n");
        printf("Enter choice: ");
        scanf("%d", &choice);

        switch (choice) {
            case 1:
                printf("Current balance: $%.2lf\n", balance);
                break;

            case 2:
                printf("Enter deposit amount: $");
                scanf("%lf", &amount);

                if (amount <= 0) {
                    printf("Invalid amount!\n");
                } else {
                    balance += amount;
                    printf("Deposited $%.2lf\n", amount);
                    printf("New balance: $%.2lf\n", balance);
                }
                break;

            case 3:
                printf("Enter withdrawal amount: $");
                scanf("%lf", &amount);

                if (amount <= 0) {
                    printf("Invalid amount!\n");
                } else if (amount > balance) {
                    printf("Insufficient funds!\n");
                    printf("Available balance: $%.2lf\n", balance);
                } else {
                    balance -= amount;
                    printf("Withdrawn $%.2lf\n", amount);
                    printf("Remaining balance: $%.2lf\n", balance);
                }
                break;

            case 4:
                printf("Thank you for using our ATM!\n");
                break;

            default:
                printf("Invalid choice!\n");
        }

    } while (choice != 4);

    return 0;
}
```

## Exercises

1. **Multiplication Table Generator**: Create a program that displays a formatted multiplication table for numbers 1-12.

2. **Palindrome Checker**: Check if a number is a palindrome (reads the same forwards and backwards). For example, 12321 is a palindrome.

3. **Armstrong Number Finder**: Find all Armstrong numbers between 1 and 1000. An Armstrong number equals the sum of its digits each raised to the power of the number of digits. (153 = 1³ + 5³ + 3³)

4. **Student Grade Manager**: Create a menu system to add student grades, calculate class average, find highest/lowest, and display grade distribution.

5. **Simple Text Adventure**: Create a text-based adventure game with multiple rooms, choices, and win/lose conditions. Use nested loops and switches for game logic.

## Key Takeaways

- Combine if-else, switch, loops, break, and continue for complex programs
- Break down problems into smaller steps
- Use appropriate control structures for each task
- Menu-driven programs benefit from do-while loops and switch statements
- Pattern printing improves understanding of nested loops
- Input validation prevents crashes and improves user experience
- Always provide clear user feedback
- Test edge cases (zero, negative, very large numbers)
- Comment complex logic for maintainability
- Practice is essential - build many small programs to master control flow
