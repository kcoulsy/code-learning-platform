# Algorithms

Welcome to the comprehensive algorithms course! This course features **11 in-depth lessons** and **2 hands-on projects** with **65 total steps**, taking you from fundamental algorithm concepts to advanced optimization techniques through practical implementation.

## Course Structure

### Foundations (Lessons 1-3)

- **Lesson 1**: Big-O Analysis (6 steps)
- **Lesson 2**: Recursion (6 steps)
- **Lesson 3**: Searching Algorithms (6 steps)

### Sorting (Lessons 4-5)

- **Lesson 4**: Basic Sorting - Bubble, Selection, Insertion (6 steps)
- **Lesson 5**: Advanced Sorting - Merge, Quick, Heap (6 steps)

### Algorithmic Paradigms (Lessons 6-8)

- **Lesson 6**: Divide and Conquer (5 steps)
- **Lesson 7**: Greedy Algorithms (6 steps)
- **Lesson 8**: Dynamic Programming (8 steps)

### Graph Algorithms (Lessons 9-10)

- **Lesson 9**: Graph Algorithms - BFS and DFS (6 steps)
- **Lesson 10**: Shortest Path Algorithms (6 steps)

### Advanced Topics (Lesson 11)

- **Lesson 11**: Backtracking and Advanced Techniques (6 steps)
- **Lesson 12**: Course Project - Pathfinding Application (10 steps)

## What You'll Learn

- **Algorithm Analysis**: Time and space complexity, Big-O notation
- **Recursion**: Base cases, recursive patterns, tree traversal
- **Searching**: Linear, binary, and advanced search techniques
- **Sorting**: Comparison sorts, divide-and-conquer, heap sort
- **Optimization**: Greedy algorithms, dynamic programming
- **Graph Algorithms**: Traversal, shortest path, topological sort
- **Problem Solving**: Backtracking, bit manipulation

## Key Features

- **C Implementation**: All algorithms implemented in C for deep understanding
- **Complexity Analysis**: Understand trade-offs between approaches
- **Step-by-Step**: Detailed explanation of each algorithm
- **Real-World Projects**: Build pathfinding and optimization applications
- **Pattern Recognition**: Learn to identify algorithmic patterns

## Prerequisites

- Completion of "Data Structures" course or equivalent knowledge
- Understanding of arrays, linked lists, trees, graphs
- Familiar with basic math (logarithms, summation)

## Recommended Timeline

This course is designed for completion in **10-14 weeks**:

- **Intensive**: 2 lessons per week (6 weeks)
- **Standard**: 1 lesson per week (11 weeks)
- **Self-Paced**: Progress at your own speed
