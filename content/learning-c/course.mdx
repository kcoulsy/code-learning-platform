# Learning C

Welcome to the complete C programming course! This comprehensive program takes you from absolute beginner to confident intermediate C programmer through a carefully designed learning roadmap.

## Course Structure

This course includes **10 in-depth lessons** and **2 hands-on projects**:

### Phase 1: Foundations

- **Lesson 1**: Introduction to C and Setup (7 steps)
- **Lesson 2**: Variables, Data Types, and Basic I/O (7 steps)
- **Lesson 3**: Operators and Expressions (5 steps)
- **Lesson 4**: Control Flow (7 steps)

### Phase 2: Functions & Pointers

- **Lesson 5**: Functions, Arrays, and Strings (8 steps)
- **Lesson 6**: Pointers (7 steps)

### Phase 3: Dynamic Memory

- **Lesson 7**: Dynamic Memory Management (8 steps)
- **Project 1**: Linked List Implementation (7 steps)

### Phase 4: Modular Code & Basics of Advanced Concepts

- **Lesson 9**: Structures and Modular Programming (8 steps, including preprocessor basics)
- **Project 2**: Contact Book Application (7 steps)

### Phase 5: Intermediate Concepts

- **Lesson 11**: Advanced C Concepts (6 steps)
- **Lesson 12**: File I/O (5 steps)

## What You'll Learn

- **Fundamentals**: Variables, data types, operators, control flow, functions
- **Memory Management**: Pointers, dynamic allocation, memory safety, debugging with valgrind
- **Data Structures**: Arrays, strings, structures, linked lists
- **Modular Programming**: Header files, separate compilation, professional code organization, preprocessor basics
- **Intermediate Concepts**: Const correctness, static keyword, bitwise operations, bit flags, error handling
- **File I/O**: Reading and writing files, file positioning, binary vs text files
- **Real-World Projects**: Build a linked list and a contact book application

## Key Features

- **Project-Based Learning**: Two comprehensive projects apply concepts in real-world scenarios
- **Safety First**: Heavy emphasis on memory safety, buffer overflow prevention, and debugging tools
- **Professional Practices**: Learn industry-standard tools (gcc, gdb, valgrind, clang-format)
- **Progressive Difficulty**: Carefully structured from beginner basics to intermediate C programming
- **Hands-On**: Numerous steps with hundreds of code examples and practice exercises

## Prerequisites

No prior programming experience required. Just bring your curiosity and willingness to learn.

## What's Next?

After completing this course, you'll have a solid foundation in C programming. To continue your journey and become an advanced C programmer, check out the **Advanced C Programming** course, which covers:

- Professional debugging and tooling (gdb, sanitizers, static analysis)
- The C preprocessor in depth
- Function pointers and callbacks
- Standard library internals
- Systems programming (POSIX, process management, IPC)
- Undefined behavior and the C memory model
- Real-world code patterns and best practices
- Advanced projects (memory allocator, data structure library, Unix shell)
