# Learning C

Welcome to the complete C programming course! This comprehensive program features **13 course units** with **88 total steps**, taking you from absolute beginner to confident systems programmer through a carefully designed 7-phase learning roadmap.

## Course Structure

This course includes **10 in-depth lessons** and **4 hands-on projects**:

### Phase 0-1: Foundations (Units 1-4)

- **Lesson 1**: Introduction to C and Setup (7 steps)
- **Lesson 2**: Variables, Data Types, and Basic I/O (7 steps)
- **Lesson 3**: Operators and Expressions (5 steps)
- **Lesson 4**: Control Flow (7 steps)

### Phase 2-3: Functions & Pointers (Units 5-6)

- **Lesson 5**: Functions, Arrays, and Strings (8 steps)
- **Lesson 6**: Pointers (7 steps)

### Phase 4: Dynamic Memory (Units 7-8)

- **Lesson 7**: Dynamic Memory Management (8 steps)
- **Project 1**: Linked List Implementation (7 steps)

### Phase 5: Modular Code (Units 9-10)

- **Lesson 9**: Structures and Modular Programming (7 steps)
- **Project 2**: Contact Book Application (7 steps)

### Phase 6-7: Advanced & Systems (Units 11-13)

- **Lesson 11**: Advanced C Concepts (8 steps)
- **Lesson 12**: Systems Programming and File I/O (7 steps)
- **Project 3A**: Unix Shell Project (5 steps)
- **Project 3B**: Custom Memory Allocator Project (4 steps)

## What You'll Learn

- **Fundamentals**: Variables, data types, operators, control flow, functions
- **Memory Management**: Pointers, dynamic allocation, memory safety, debugging with valgrind
- **Data Structures**: Arrays, strings, structures, linked lists
- **Modular Programming**: Header files, separate compilation, professional code organization
- **Advanced Concepts**: Const correctness, static keyword, bitwise operations, function pointers, callbacks
- **Systems Programming**: File I/O, POSIX basics, process management, inter-process communication
- **Real-World Projects**: Build a linked list, contact book application, a Unix shell, and a custom memory allocator

## Key Features

- **Project-Based Learning**: Three comprehensive projects apply concepts in real-world scenarios
- **Safety First**: Heavy emphasis on memory safety, buffer overflow prevention, and debugging tools
- **Professional Practices**: Learn industry-standard tools (gcc, gdb, valgrind, clang-format, make)
- **Progressive Difficulty**: Carefully structured from beginner basics to advanced systems programming
- **Hands-On**: 88 steps with hundreds of code examples and practice exercises

## Prerequisites

No prior programming experience required. Just bring your curiosity and willingness to learn.

## Recommended Timeline

This course is designed for completion in **7-13 weeks**:

- **Intensive**: 1-2 units per week (7 weeks)
- **Standard**: 1 unit per week (13 weeks)
- **Self-Paced**: Progress at your own speed
