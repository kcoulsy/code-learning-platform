# Basic Data Types in C

C has several fundamental data types for storing different kinds of values.

## Integer Types

```c
int number = 42;           // Standard integer
short small = 100;         // Smaller integer
long big = 1000000L;       // Larger integer
long long huge = 9223372036854775807LL; // Very large
```

## Floating-Point Types

```c
float price = 19.99f;      // Single precision (6-7 digits)
double precise = 3.14159265358979; // Double precision (15-16 digits)
```

## Character Type

```c
char letter = 'A';         // Single character
char newline = '\n';       // Escape character
```

## Size of Data Types

Use `sizeof()` to check the size in bytes:

```c
#include <stdio.h>

int main() {
    printf("int: %lu bytes\n", sizeof(int));
    printf("float: %lu bytes\n", sizeof(float));
    printf("double: %lu bytes\n", sizeof(double));
    printf("char: %lu bytes\n", sizeof(char));
    return 0;
}
```

## Type Modifiers

- `signed` - can hold negative values (default for int)
- `unsigned` - only positive values, doubles the range

```c
unsigned int positive = 4294967295; // 0 to 4,294,967,295
signed int any = -2147483648;       // -2,147,483,648 to 2,147,483,647
```

## Format Specifiers for printf

Each data type has a corresponding format specifier for printing:

```c
#include <stdio.h>

int main(void) {
    int age = 25;
    float price = 19.99f;
    double pi = 3.14159265358979;
    char grade = 'A';
    unsigned int count = 100u;

    printf("Integer: %d\n", age);
    printf("Float: %f\n", price);
    printf("Double: %lf\n", pi);
    printf("Character: %c\n", grade);
    printf("Unsigned: %u\n", count);

    return 0;
}
```

### Common Format Specifiers

| Type | Specifier | Example |
|------|-----------|---------|
| `int` | `%d` or `%i` | `printf("%d", 42)` |
| `unsigned int` | `%u` | `printf("%u", 100u)` |
| `long` | `%ld` | `printf("%ld", 1000L)` |
| `long long` | `%lld` | `printf("%lld", 999LL)` |
| `float` | `%f` | `printf("%f", 3.14f)` |
| `double` | `%lf` | `printf("%lf", 3.14)` |
| `char` | `%c` | `printf("%c", 'A')` |
| `char*` (string) | `%s` | `printf("%s", "Hello")` |
| `void*` (pointer) | `%p` | `printf("%p", ptr)` |
| `size_t` | `%zu` | `printf("%zu", sizeof(int))` |

### Format Specifier Modifiers

```c
#include <stdio.h>

int main(void) {
    float pi = 3.14159265f;
    int num = 42;

    // Width and precision
    printf("%.2f\n", pi);        // 3.14 (2 decimal places)
    printf("%.5f\n", pi);        // 3.14159 (5 decimal places)
    printf("%10d\n", num);       // "        42" (width 10, right-aligned)
    printf("%-10d\n", num);      // "42        " (width 10, left-aligned)
    printf("%05d\n", num);       // "00042" (width 5, zero-padded)

    return 0;
}
```

## Data Type Ranges

```c
#include <stdio.h>
#include <limits.h>
#include <float.h>

int main(void) {
    printf("int: %d to %d\n", INT_MIN, INT_MAX);
    printf("unsigned int: 0 to %u\n", UINT_MAX);
    printf("char: %d to %d\n", CHAR_MIN, CHAR_MAX);
    printf("long: %ld to %ld\n", LONG_MIN, LONG_MAX);

    printf("float max: %e\n", FLT_MAX);
    printf("double max: %e\n", DBL_MAX);

    return 0;
}
```

## Common Pitfalls

### Wrong Format Specifier

```c
// Warning: using wrong format specifier
int x = 42;
printf("%f", x);  // WRONG: %f expects double, got int
printf("%d", x);  // CORRECT

float y = 3.14f;
printf("%d", y);  // WRONG: %d expects int, got float
printf("%f", y);  // CORRECT
```

Using the wrong format specifier can cause:
- Garbage output
- Undefined behavior
- Security vulnerabilities

### Integer Overflow

```c
#include <stdio.h>
#include <limits.h>

int main(void) {
    int max = INT_MAX;  // 2,147,483,647
    printf("max = %d\n", max);
    printf("max + 1 = %d\n", max + 1);  // Overflow! Wraps to INT_MIN

    return 0;
}
```

## Key Takeaways

- C has fundamental types: `int`, `float`, `double`, `char`
- Use size modifiers: `short`, `long`, `long long`
- Use sign modifiers: `signed` (default), `unsigned`
- Always use the correct format specifier in `printf`
- Check `<limits.h>` and `<float.h>` for type ranges
- Be aware of integer overflow and underflow
- Use `sizeof()` to determine type sizes on your system
