# Singly Linked Lists

A **singly linked list** has nodes with a single pointer to the next node. This is the simplest form of linked list.

## Basic Operations

### Insert at Beginning

```c
Node *insertAtBeginning(Node *head, int value) {
    Node *newNode = createNode(value);
    newNode->next = head;
    return newNode;  // New node becomes head
}
```

### Insert at End

```c
Node *insertAtEnd(Node *head, int value) {
    Node *newNode = createNode(value);

    if (head == NULL) {
        return newNode;  // Empty list
    }

    Node *current = head;
    while (current->next != NULL) {
        current = current->next;
    }
    current->next = newNode;
    return head;
}
```

### Insert at Position

```c
Node *insertAtPosition(Node *head, int value, int pos) {
    if (pos < 1) {
        return insertAtBeginning(head, value);
    }

    Node *newNode = createNode(value);
    Node *current = head;

    for (int i = 1; i < pos - 1 && current != NULL; i++) {
        current = current->next;
    }

    if (current == NULL) {
        free(newNode);
        return head;
    }

    newNode->next = current->next;
    current->next = newNode;
    return head;
}
```

### Traversal

```c
void printList(Node *head) {
    Node *current = head;
    while (current != NULL) {
        printf("%d -> ", current->data);
        current = current->next;
    }
    printf("NULL\n");
}
```

### Search

```c
int search(Node *head, int target) {
    Node *current = head;
    int position = 0;

    while (current != NULL) {
        if (current->data == target) {
            return position;
        }
        current = current->next;
        position++;
    }
    return -1;  // Not found
}
```
